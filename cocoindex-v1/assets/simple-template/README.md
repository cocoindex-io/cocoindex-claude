# Simple CocoIndex Template

This template provides a minimal starting point for a CocoIndex v1 project.

## Usage

Copy this template to your project directory and customize as needed. The template includes:

- Basic file transformation pipeline structure
- Proper pyproject.toml configuration for CocoIndex v1 pre-release
- Environment configuration template

## Customization

1. Update `pyproject.toml` with your project name and dependencies
2. Modify `main.py` to implement your transformation logic
3. Configure `.env` with your database paths and API keys
